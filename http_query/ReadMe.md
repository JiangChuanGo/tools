1. clone this repo
  ```bash
  git clone https://github.com/JiangChuanGo/tools.git 
  ```

2. pip install requirements.txt
  ```bash
  pip install -r requirements.txt
  ```

3. run the too
  ```bash
  python main.py
  ```
